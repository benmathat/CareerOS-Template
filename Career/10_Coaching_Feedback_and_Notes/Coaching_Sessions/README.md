# Coaching Sessions

This folder contains notes from career coaching sessions.

## Purpose

Document insights and action items from coaching:
- Session notes and key takeaways
- Advice and recommendations received
- Action items and commitments
- Progress tracking
- Resources and tools shared

## How to Use

- Create one file per coaching session
- Document date, coach name, and key topics
- Capture insights, advice, and action items
- Track progress on commitments
- Review before next session

This folder helps you maximize value from coaching and track your growth.
