# Resume Feedback

This folder contains feedback received on your resumes.

## Purpose

Capture and act on resume feedback:
- Feedback from recruiters, hiring managers, or coaches
- ATS optimization suggestions
- Formatting and design feedback
- Content and messaging feedback
- Action items and improvements

## How to Use

- Save feedback documents and notes
- Track who provided feedback and when
- Note patterns across multiple sources
- Create action items for improvements
- Reference when updating resumes

This folder ensures you incorporate feedback systematically and improve over time.
