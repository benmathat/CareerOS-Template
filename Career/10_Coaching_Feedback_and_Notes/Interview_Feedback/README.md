# Interview Feedback

This folder contains feedback received after interviews.

## Purpose

Capture and learn from interview feedback:
- Post-interview feedback from interviewers
- Rejection feedback and reasons
- Areas for improvement
- What went well
- Action items for future interviews

## How to Use

- Document feedback immediately after receiving it
- Look for patterns across multiple interviews
- Identify specific areas to improve
- Update interview prep materials based on feedback
- Track improvement over time

This folder helps you learn from each interview and improve systematically.
