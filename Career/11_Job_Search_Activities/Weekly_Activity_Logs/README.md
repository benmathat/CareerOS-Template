# Weekly Activity Logs

**Purpose:**  
Weekly summaries of job search activities for unemployment documentation and personal accountability.

**Use this when:**
- Filing weekly unemployment claims
- Reviewing your search activity levels
- Providing proof of job search efforts

---

## How to Use

1. Create a new file for each week using the naming format: `week-YYYY-MM-DD.md` (use the Monday date of the week)
2. Use the `weekly-activity-log-template.md` as a starting point
3. Log activities daily as they happen
4. Review and complete the summary before filing your unemployment claim

---

## File Naming Convention

- Format: `week-YYYY-MM-DD.md`
- Example: `week-2026-01-20.md` (for the week starting January 20, 2026)
- Use the Monday date of the week being documented

---

## Quick Reference

- **Minimum activities:** Check your state's unemployment requirements
- **Documentation:** Include dates, company names, role titles, and contact information
- **Consistency:** Log activities daily to avoid gaps
