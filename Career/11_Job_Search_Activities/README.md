# Job Search Activities — Texas Workforce Commission (TWC) Compliance

**Location:** `Career/11_Job_Search_Activities/`

**Purpose:**  
Documents work search activities in compliance with Texas Workforce Commission (TWC) unemployment eligibility requirements. This folder provides an audit-safe, ready-to-use system for tracking and certifying job search activities.

This folder serves three key functions:
1. **Compliance:** Meets TWC requirements for documenting work search activities
2. **Audit Protection:** Provides detailed, contemporaneous records that protect you in case of disputes or audits
3. **Accountability:** Maintains a clear record of your search efforts and progress

**Use this when:**
- Filing weekly unemployment certifications with TWC
- Documenting required work search activities
- Tracking your search activity levels
- Providing proof of work search efforts if audited
- Preparing for potential TWC investigations

---

## What Goes Here

- **Work Search Log:** Canonical record of all work search activities (`Work_Search_Log.md`)
- **Weekly Certifications:** Snapshots of what was submitted to TWC each week
- **Evidence:** Supporting documentation (application confirmations, emails, interview invites)

---

## Key Principles

1. **Document Immediately:** Record activities as they happen, not at the end of the week
2. **Be Specific:** Include dates, company names, role titles, contact information, and outcomes
3. **Over-Comply:** This format exceeds TWC requirements in a good way — it reads like a professional compliance record
4. **Maintain Evidence:** Save proof of activities (confirmations, emails, screenshots) in the Evidence folder

---

## Texas Workforce Commission Requirements

TWC requires you to track and document:

| Requirement | Where It's Covered |
|------------|-------------------|
| Date of activity | Date column in `Work_Search_Log.md` |
| Type of work search | Activity Type column |
| Employer/contact | Employer / Platform column |
| Method of contact | Method column |
| Result of activity | Outcome / Status column |
| Proof upon request | Evidence folder |
| Weekly compliance | Weekly Certification Summary |

**Why This Format Passes Audits:**
- No vague language
- No lumped activities
- Clear weekly grouping
- Declarative statement included
- Evidence references (critical for audits)

## How the Pieces Work Together

**`Work_Search_Log.md`**
- Canonical record of all activities
- Source of truth if audited
- Updated daily as activities happen
- Contains all required TWC elements

**`Weekly_Certifications/`**
- Snapshot per week
- Mirrors what you submit to TWC
- References rows from `Work_Search_Log.md`
- Creates clean audit trail

**`Evidence/`**
- Application confirmations
- Email screenshots
- Interview invitations
- Any proof of activities

This separation keeps:
- Compliance clean
- Narrative intact
- Evidence traceable

## Recommended Daily Workflow (5–7 minutes)

1. **Do activity** (apply, network, research)
2. **Immediately add a row** to `Work_Search_Log.md`
3. **Drop any proof** into `/Evidence/` folder
4. **End of week:**
   - Count activities
   - Fill in Weekly Certification Summary
   - Create weekly certification file
   - Submit to TWC
