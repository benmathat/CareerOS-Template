# Weekly Certifications

**Purpose:**  
Weekly snapshots of work search activities submitted to Texas Workforce Commission (TWC).

**Use this when:**
- Filing weekly unemployment certifications
- Creating a record of what was submitted to TWC
- Referencing past certifications for audit purposes

---

## How to Use

1. Create a new file for each week using the naming format: `YYYY-W##.md` (e.g., `2026-W03.md`)
2. Copy relevant entries from `Work_Search_Log.md` for that week
3. Include the summary that matches what you submitted to TWC
4. Reference any supporting evidence files

---

## File Naming Convention

- Format: `YYYY-W##.md`
- Example: `2026-W03.md` (for week 3 of 2026)
- Use ISO week numbering (week starts Monday)

---

## Weekly Certification Template

Each weekly file should include:
- Week dates (Monday to Sunday)
- Activities from `Work_Search_Log.md` for that week
- Total activity count
- Certification submission date
- Reference to evidence files if applicable

---

## Integration

- **Source:** Entries come from `Work_Search_Log.md`
- **Evidence:** Reference files in `/Evidence/` folder
- **Purpose:** Creates a clean audit trail of what was submitted to TWC
