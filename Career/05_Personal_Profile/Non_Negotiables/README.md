# Non-Negotiables

This folder defines your absolute requirements and deal-breakers.

## Purpose

Clarify what you will not compromise on:
- Absolute requirements (compensation minimums, location, etc.)
- Deal-breakers (red flags, unacceptable conditions)
- Values you won't violate
- Conditions that would cause you to walk away

## How to Use

- Create a clear list of non-negotiables
- Keep it concise and specific
- Reference before accepting any offer
- Update as your situation changes
- Use to filter opportunities early

This folder protects you from accepting roles that will make you unhappy.
