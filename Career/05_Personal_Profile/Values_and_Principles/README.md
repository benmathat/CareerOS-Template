# Values and Principles

This folder captures your core values and guiding principles.

## Purpose

Document what matters most to you professionally:
- Core values and beliefs
- Principles that guide decisions
- What you stand for
- What you won't compromise on

## How to Use

- Create a values document that articulates your principles
- Reference when evaluating company culture fit
- Use to guide decisions during interviews and negotiations
- Review periodically to ensure alignment with current priorities

This folder helps you identify good-fit opportunities and avoid misaligned roles.
