# Work Preferences

This folder documents your work style preferences and requirements.

## Purpose

Define how you work best:
- Remote, hybrid, or in-office preferences
- Preferred work hours and schedule flexibility
- Team size and structure preferences
- Management style preferences
- Communication preferences
- Work environment needs

## How to Use

- Create a document outlining your preferences
- Distinguish between "must-haves" and "nice-to-haves"
- Reference when evaluating opportunities
- Use to ask informed questions during interviews
- Update as preferences evolve

This folder helps you identify roles that match how you work best.
