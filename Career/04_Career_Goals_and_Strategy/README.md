# Direction & Strategy

This folder defines **where you are going and what you are optimizing for**.

It exists to:
- Clarify target roles and trajectories
- Define constraints (compensation, lifestyle, location)
- Prevent reactive or panic-driven decisions

## How to Use
- Review before applying to roles
- Update when priorities change
- Use as a filter when evaluating opportunities

If this folder is empty or outdated, job seeking will feel chaotic.
