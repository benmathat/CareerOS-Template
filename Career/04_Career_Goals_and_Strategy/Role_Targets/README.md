# Role Targets

This folder defines the specific roles and positions you are targeting.

## Purpose

Clarify what roles you're pursuing:
- Target job titles and levels
- Role descriptions and requirements
- Multiple role options (primary, secondary, exploratory)
- Role-specific goals and expectations

## How to Use

- Create one file per target role type
- Include key requirements and qualifications
- Note alignment with your experience and skills
- Reference when evaluating job postings
- Update as your targets evolve

This folder helps you stay focused and evaluate opportunities against clear criteria.
