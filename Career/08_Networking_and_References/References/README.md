# References

This folder contains your reference list and context for each reference.

## Purpose

Maintain your reference list and context:
- List of potential references
- Contact information and relationship context
- What each reference can speak to
- When and how to use each reference
- Reference check preparation

## How to Use

- Create a master reference list
- Document each reference's relationship to you
- Note what they can attest to (skills, projects, character)
- Keep contact information current
- Prepare references before they're contacted
- Thank references after use

This folder ensures you have strong, prepared references ready when needed.
