# Informational Interviews

This folder contains notes from informational interviews and conversations.

## Purpose

Capture insights from informational interviews:
- Notes from conversations with people in your target roles
- Industry insights and advice
- Company culture information
- Career path guidance
- Referral opportunities

## How to Use

- Create one file per informational interview
- Include date, contact information, and key topics discussed
- Document insights, advice, and next steps
- Track follow-up actions and relationship maintenance
- Reference when preparing for similar roles or companies

This folder helps you learn from others' experiences and build relationships.
