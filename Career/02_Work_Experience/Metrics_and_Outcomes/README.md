# Metrics and Outcomes

This folder captures quantifiable achievements and measurable impact.

## Purpose

Document metrics, KPIs, and outcomes that demonstrate your impact:
- Performance metrics (e.g., "reduced latency by 40%")
- Business outcomes (e.g., "increased revenue by $2M")
- Scale metrics (e.g., "managed 50-person team")
- Efficiency gains (e.g., "reduced costs by 25%")
- User/engagement metrics (e.g., "improved NPS from 7 to 9")

## How to Use

- Organize by role, project, or time period
- Include context for each metric (baseline, timeframe, method)
- Reference this folder when crafting resume bullets or interview stories
- Keep raw numbers here; polished versions go in resumes

Numbers tell a story. This folder ensures you never forget the details.
