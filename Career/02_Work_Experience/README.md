# Raw Career Data

This folder is your **career memory**.

Its purpose is to capture:
- Roles and employers
- Projects and initiatives
- Achievements and failures
- Metrics, outcomes, and impact

## How This Folder Is Used
- Source material for resumes
- Input for interview stories
- Reference for reflection and growth

## Important Notes
- This folder is intentionally verbose
- Nothing is removed for being “too much”
- Accuracy and completeness matter more than polish

This folder should grow continuously over your career.
