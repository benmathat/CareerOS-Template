# Roles

This folder contains detailed descriptions of your roles and positions.

## Purpose

Capture comprehensive information about each role you've held, including:
- Role title and level
- Company and team context
- Timeframe and duration
- Key responsibilities
- Reporting structure
- Team size and composition

## How to Use

- Create one file per role (e.g., `senior-engineer-acme-2020-2023.md`)
- Include full context that might be trimmed from resumes
- Reference this folder when updating resumes or preparing interview stories

This complements the `Projects/` folder by providing role-level context.
