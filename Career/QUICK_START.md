# Quick Start Guide

Welcome to your Career Workspace! This guide will help you get started in **30-60 minutes** with the minimum setup needed to begin a job search.

## Before You Start

**Time needed:** 30-60 minutes  
**Goal:** Get the essential foundation in place so you can start capturing opportunities and applying

**Don't worry about:**
- Filling everything out perfectly
- Completing all folders
- Making it perfect

**Do focus on:**
- Getting the basics in place
- Starting to capture opportunities
- Building momentum

---

## The First 5 Things to Populate

Complete these in order. Each builds on the previous one.

### 1. Set Your Current Focus (5 minutes)

**File:** `00_ReadMe_and_Index/career-index.md`

**What to do:**
- Fill in the "Snapshot" section:
  - Current Focus: What are you doing right now? (e.g., "Active job search for software engineering roles")
  - Primary Target Roles: What roles are you targeting? (e.g., "Senior Software Engineer")
  - Geographic / Remote Preference: (e.g., "Remote or San Francisco Bay Area")
  - Compensation Target: Your target range (e.g., "$150k-$180k base")

**Why:** This is your control center. It helps you stay focused and prevents drift.

---

### 2. Define Your Target Role (10-15 minutes)

**File:** `04_Career_Goals_and_Strategy/Role_Targets/role-target-template.md`

**What to do:**
1. Copy the template: `04_Career_Goals_and_Strategy/Role_Targets/role-target-template.md`
2. Save as: `04_Career_Goals_and_Strategy/Role_Targets/my-primary-target.md`
3. Fill in:
   - Role title and level
   - Key responsibilities you're targeting
   - Required skills (what you have vs. what you need)
   - Why this role fits you

**Why:** Knowing what you're targeting helps you evaluate opportunities and tailor applications.

**Template:** `04_Career_Goals_and_Strategy/Role_Targets/role-target-template.md`

---

### 3. Set Up Your Application Tracker (5 minutes)

**File:** `07_Applications_and_Interviews/Applications/application-tracker.md`

**What to do:**
- Open the tracker
- Familiarize yourself with the table structure
- You'll add rows as you find opportunities

**Why:** This is where you'll track everything. Having it ready means you can log opportunities immediately.

**Template:** Already set up at `07_Applications_and_Interviews/Applications/application-tracker.md`

---

### 4. Document Your Work Preferences (10 minutes)

**File:** `05_Personal_Profile/Work_Preferences/work-preferences-template.md`

**What to do:**
1. Copy the template: `05_Personal_Profile/Work_Preferences/work-preferences-template.md`
2. Save as: `05_Personal_Profile/Work_Preferences/my-preferences.md`
3. Fill in:
   - Location/remote preferences
   - Schedule preferences
   - Must-haves vs. nice-to-haves
   - Deal-breakers

**Why:** This helps you quickly evaluate opportunities and avoid bad-fit roles.

**Template:** `05_Personal_Profile/Work_Preferences/work-preferences-template.md`

---

### 5. Initialize Your Search Status (5 minutes)

**File:** `00_ReadMe_and_Index/search-status.md`

**What to do:**
- Mark your current phase (Exploration, Active Applications, etc.)
- Set 1-3 weekly priorities
- Leave the rest for now

**Why:** This keeps you focused week-to-week and prevents overwhelm.

**Template:** Already set up at `00_ReadMe_and_Index/search-status.md`

---

## Minimum Viable Setup Checklist

After completing the first 5 items above, you have the **minimum viable setup**. You can now:

- ✅ Start capturing opportunities
- ✅ Evaluate roles against your criteria
- ✅ Track applications
- ✅ Stay focused on priorities

**Optional but recommended next steps:**
- Set up your master resume location (if you have one)

## Common First-Time Scenarios

### "I'm starting a job search right now"

**Do this:**
1. Complete the First 5 Things (above)
2. Start capturing opportunities in `06_Job_Opportunities/`
3. When you find something interesting, research it using the company research template
4. When you apply, create an application detail file

**Time:** 30-45 minutes to get started, then ongoing

---

### "I want to explore but not apply yet"

**Do this:**
1. Complete items 1, 2, and 4 from First 5 Things (focus, role target, preferences)
2. Start saving interesting opportunities to `06_Job_Opportunities/Job_Postings/`
3. Research companies using the company research template
4. Update your role targets as you learn more

**Time:** 20-30 minutes, then ongoing exploration

---

### "I already have opportunities I'm tracking"

**Do this:**
1. Complete the First 5 Things (quick version)
2. Immediately log your existing opportunities:
   - Add to `07_Applications_and_Interviews/Applications/application-tracker.md`
   - Create application detail files for each using the template
3. Set up your search status

**Time:** 45-60 minutes to get everything logged

## Getting Help

### Understanding the System
- **Overview:** Read `README.md` in the root `Career/` folder
- **Workflows:** See `WORKFLOWS.md` for how everything connects
- **Folder details:** Each folder has a `README.md` explaining its purpose

### When You're Stuck
- Ask: **"Is this thinking, doing, or learning?"**
  - Thinking → `04_Career_Goals_and_Strategy/` or `05_Personal_Profile/`
  - Doing → `07_Applications_and_Interviews/` or `06_Job_Opportunities/`
  - Learning → `10_Coaching_Feedback_and_Notes/` or `09_Research_and_Market_Intelligence/`

### Key Principle
**Start simple, add complexity as needed.** The system is designed to grow with you. You don't need to fill everything out immediately.
