## Purpose

This template helps you track detailed information for each job application, from initial interest through final outcome. Use it to maintain context, prepare for interviews, capture feedback, and learn from each application experience.

- When you decide to apply to a role (create a new file for each application)
- Before each interview (use pre-interview checklist)
- Immediately after each interview (capture notes while fresh)
- When evaluating offers (use decision matrix)
