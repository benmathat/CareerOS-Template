# Offers and Negotiation

This folder tracks job offers and negotiation details.

## Purpose

Document:
- Offer details (compensation, benefits, equity, etc.)
- Negotiation conversations and strategies
- Counter-offers
- Decision-making process
- Comparison between multiple offers

## How to Use

- Create one file per offer (e.g., `company-role-offer-2024.md`)
- Include all offer components: base, bonus, equity, benefits, PTO, etc.
- Document negotiation timeline and outcomes
- Reference your compensation targets from `04_Career_Goals_and_Strategy/`
- Keep notes on what worked and what didn't in negotiations

This folder helps you:
- Make informed decisions
- Learn from each negotiation
- Build leverage for future negotiations
- Track market rates over time
