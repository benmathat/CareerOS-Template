# Online Profiles

This folder contains text and content for online professional profiles.

## Purpose

Store content for:
- LinkedIn profile sections
- GitHub profile/README
- Personal website content
- Portfolio site text
- Other professional platform profiles

## How to Use

- Keep profile text versions here (not just links)
- Track what's currently live vs. draft versions
- Ensure consistency with `Resume_Master/` content
- Update when you refresh your online presence

Your online profiles should align with your resume but can be more conversational and detailed.
