# Cover Letters

This folder contains cover letters and application letters.

## Purpose

Store cover letters tailored to specific applications:
- Job-specific cover letters
- Referral letters
- Networking outreach letters
- Follow-up letters

## How to Use

- Name files clearly: `company-role-date.md` or `company-role.pdf`
- Reference the specific job posting and company research
- Link to the corresponding application in `07_Applications_and_Interviews/Applications/`
- Keep both draft and final versions if helpful

Cover letters should demonstrate alignment between your experience (from `02_Work_Experience/`) and the role's requirements.
