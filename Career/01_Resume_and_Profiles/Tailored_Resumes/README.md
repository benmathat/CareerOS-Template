# Tailored Resumes

This folder contains job-specific resume versions.

## Purpose

Store tailored resumes created for specific roles or companies:
- Role-specific versions highlighting relevant experience
- Industry-specific positioning
- ATS-optimized versions
- Different length variants (1-page, 2-page, etc.)

## How to Use

- Name files clearly: `company-role-date.md` or `company-role.pdf`
- Always start from `Resume_Master/` as the source of truth
- Note what was emphasized or de-emphasized for each version
- Keep track of which version was used for which application

Remember: This folder is for **positioning**, not inventing experience. All content should trace back to `02_Work_Experience/`.
