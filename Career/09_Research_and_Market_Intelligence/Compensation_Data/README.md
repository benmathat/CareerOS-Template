# Compensation Data

This folder contains salary benchmarks and compensation research.

## Purpose

Research and track compensation information:
- Salary ranges by role, level, and location
- Total compensation benchmarks
- Equity and benefits information
- Negotiation data and strategies
- Market rate sources and tools

## How to Use

- Research compensation for your target roles
- Document data sources and dates
- Track ranges by location and company size
- Use to set realistic expectations
- Reference during offer evaluation and negotiation

This folder supports informed negotiation and helps you value yourself accurately.
